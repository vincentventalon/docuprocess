# Content Extraction Guide

## What to Extract from Each Page Type

---

## Homepage

### Key Elements

**Hero Section**
- Primary headline (H1)
- Subheadline/tagline
- Primary CTA text and destination
- Secondary CTA (if present)

**Value Proposition**
- Key benefits listed (usually 3-4)
- Supporting descriptions
- Icons/imagery used

**Social Proof**
- Customer logos (list company names)
- Testimonials (quote + attribution)
- Statistics shown ("10,000+ customers", "99.9% uptime")
- Media mentions ("As seen in...")

**Feature Highlights**
- Which features are promoted on homepage
- How they're positioned
- What screenshots/demos are shown

**CTA Strategy**
- Above-fold CTA
- End-of-section CTAs
- Footer CTA
- CTA copy patterns

---

## Pricing Page

### Pricing Structure

**Model Type**
- Per seat/user
- Usage-based
- Flat rate
- Tiered features
- Hybrid

**Tiers to Document**
For each tier:
- Name
- Price (monthly and annual if both shown)
- Tagline/description
- Target audience
- Key features included
- Limits (users, usage, storage, etc.)
- What's NOT included

**Additional Elements**
- Annual discount percentage
- Enterprise/custom option
- Free tier or trial details
- Add-ons or extras
- Payment methods accepted

### Trust Elements
- Money-back guarantee
- "No credit card required"
- Security badges
- FAQ content

### Comparison Table
- Feature comparison across tiers
- Which features are used to differentiate
- Enterprise-only features

---

## Feature/Product Pages

### For Each Feature Page

**Positioning**
- Feature name
- Primary benefit headline
- Supporting description
- Target use case

**Functionality**
- What it does (capabilities)
- How it works (workflow)
- Technical details
- Limitations mentioned

**Evidence**
- Screenshots
- Demo videos
- Use case examples
- Customer quotes

**Integration Context**
- How it connects to other features
- Third-party integrations

---

## Blog

### Content Inventory

**Volume & Frequency**
- Total post count (estimate from pagination)
- Publication frequency
- Date range (oldest to newest)

**Categories/Topics**
- Category names
- Post distribution across categories
- Primary focus areas

**Content Types**
- How-to guides
- List posts
- Case studies
- Industry news
- Product updates
- Thought leadership

**SEO Signals**
- Title patterns (keywords targeted)
- URL structure
- Meta description patterns
- Internal linking strategy

**Top Content**
- Featured/pinned posts
- Posts linked from homepage
- Posts with high engagement (comments, shares if visible)

---

## Documentation

### Structure Analysis

**Organization**
- Main sections/categories
- Depth of hierarchy
- Navigation approach

**Content Types**
- Getting started guides
- Feature documentation
- API reference
- Tutorials
- Troubleshooting
- FAQ

**Quality Indicators**
- Completeness (are features documented?)
- Currency (last updated dates)
- Examples included
- Code samples (if applicable)
- Screenshots/GIFs

**Developer Focus**
- API documentation presence
- SDK/library docs
- Webhook documentation
- Integration guides

---

## Free Tools

### For Each Tool

**Tool Details**
- Name
- URL
- One-line description
- Category (calculator, generator, checker, etc.)

**Functionality**
- What input it requires
- What output it provides
- Value it delivers

**Lead Capture**
- Email required? (before/after result)
- Account required?
- Usage limits (free vs. paid)

**SEO Value**
- Keyword it targets
- Search volume potential
- Link magnet potential

---

## Solutions/Use Case Pages

### Per Page

**Target Segment**
- Industry
- Role/persona
- Company size
- Use case

**Positioning**
- How product is positioned for this segment
- Specific pain points addressed
- Tailored benefits

**Evidence**
- Segment-specific testimonials
- Relevant case studies
- Industry stats

**CTA**
- Segment-specific offer or demo

---

## Case Studies

### Per Case Study

**Company Profile**
- Company name
- Industry
- Size (employees, revenue if mentioned)
- Location

**Story Arc**
- Challenge/problem
- Solution (how product was used)
- Results (quantified if possible)

**Proof Points**
- Metrics achieved
- Quotes from customer
- Before/after comparison

**Product Usage**
- Features highlighted
- Integration with other tools
- Use case demonstrated

---

## About/Company Pages

### Company Info

**Basics**
- Founded year
- Location(s)
- Team size
- Mission/vision statement

**Leadership**
- Key executives
- Founder backgrounds

**Traction**
- Customer count
- Revenue (if public)
- Funding raised
- Notable investors

**Culture**
- Values stated
- Hiring signals
- Remote/office info

---

## Trust & Social Proof Inventory

### Collect All Instances

**Customer Logos**
- Company names
- Logo placement locations
- Categories (enterprise, startup, industry)

**Testimonials**
- Quote text
- Person name
- Title
- Company
- Photo included?

**Case Studies**
- Company name
- Results summary
- Industry

**Stats**
- Metric type (customers, users, uptime, etc.)
- Number
- Context

**Awards/Recognition**
- G2 badges
- Industry awards
- Certifications

**Media Mentions**
- Publications
- Context (review, feature, etc.)
