## Base URL


## Authentication

All requests require an API key passed in the header.

```bash
x-api-key: YOUR_API_KEY
```

## Endpoints

### POST /v1/example

Example endpoint.

```bash
curl -X POST https://api.parsedocu.com/v1/example \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"key": "value"}'
```

```json
{

}
```
